subject to minimum rest time constraints.

### Implemented Methods
- Greedy assignment heuristic
- Tabu Search metaheuristic

### Repository Structure
- `code/iteration.R`: Main R script containing all algorithms
- `data/`: Excel input files used in the experiments

### Reproducibility
All computational results reported in the study were obtained using the code
provided in this repository.
